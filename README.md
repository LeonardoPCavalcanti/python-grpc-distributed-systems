# Distributed Systems — DCA/UFRN

Coursework and experiments in distributed systems, covering RPC paradigms, gRPC with Protobuf, and service communication patterns.

## Topics

- **atividade2/** — Protocol Buffers (Protobuf) and gRPC service definition
- **rpc_variacoes/** — Variations of RPC patterns: synchronous, asynchronous, and multi-service communication
- **twitter/** — Distributed publish-subscribe simulation

## Tech Stack

- Python
- gRPC / Protobuf
- Docker (for multi-container service orchestration)
